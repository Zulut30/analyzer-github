# RepoScope

Простое веб-приложение для анализа публичных GitHub репозиториев через Timeweb AI Agent / Grok Code Fast.

## Что делает

- принимает ссылку на GitHub репозиторий;
- собирает README, языки, дерево файлов и важные манифесты через GitHub API;
- отправляет сжатый контекст в Timeweb agent;
- показывает понятное русскоязычное резюме: назначение, библиотеки, архитектуру, запуск, ключевые файлы, слабые места, оценку по 10-балльной шкале и подробный итог;
- поддерживает светлую и темную темы;
- сохраняет историю анализов в браузере пользователя на отдельной странице `/history`;
- генерирует профессиональный prompt для LLM/coding agent, чтобы исправлять найденные слабые места;
- экспортирует анализ в `txt`, `md`, `pdf` и `docx`.

## Запуск

```bash
npm install
npm start
```

По умолчанию приложение запускается на `http://localhost:3000`.

## Переменные окружения

Скопируйте `.env.example` в `.env` при необходимости.

- `TIMEWEB_OPENAI_BASE_URL` - OpenAI-compatible URL агента.
- `TIMEWEB_NATIVE_CALL_URL` - fallback endpoint `/call`.
- `TIMEWEB_API_TOKEN` - опционально, если агент требует Bearer-токен.
- `GITHUB_TOKEN` - опционально, повышает лимиты GitHub API.
